# 📊 COVID Data Analysis Dashboard (Tableau)

This project analyzes COVID-19 data using SQL and visualizes insights using Tableau.

---

## 🔧 Tools & Technologies
- MySQL
- Tableau Public
- Excel (for export)

---

- Global total cases and total deaths
- Death percentage across countries
- Population infection rate by country
- Trend of infection over time
